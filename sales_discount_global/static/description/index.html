Sales Discount with Journal Entry

Apply global discount in Sale & Invoice with journal entries

This module is provide discount journal entry in Invoice.
  • ☑   Module to manage apply discount in sale order & invoice.
  • ☑   Apply global discount by Fixed Price or Percentage.
  • ☑   Discount display in journal entry with configured account.
  • ☑   Support Multi Currency feature for applied discount.

Process

Configure Global Discount.
  • ☑   Select Discount Account: Invoicing > Configuration > Settings > Discount Account.
  • ☑   Create sale order and apply discount by fixed price or percentage.
  • ☑   Create invoice from sale order auto fill-up the discount details.
  • ☑   OR You can create new invoice and apply discount..
  • ☑   Display journal entry in invoice Other Info tab Journal Entry.
  • ☑   You can see the journal entry with configured account with discount amount.
  • ☑   Menu: Invoicing > Adviser > Journal Entries.
  • ☑   Support Multi Currency feature for applied discount.
  • ☑   Invoice with set different currency & effect on Journal Entry.
